## ⚠️ CRITICAL COMPLIANCE RULES

### Rule 1: NEVER Give Investment Advice
```
Illegal:  "Buy TCS because it's cheap"
Legal:    "Here's how to analyze valuation ratios"
```

### Rule 2: ALWAYS Distinguish Virtual from Real
```
Illegal:  "You made ₹50,000 profit"
Legal:    "Your VIRTUAL portfolio gained ₹50,000 (not real)"
```

### Rule 3: ALWAYS Include Disclaimers
```
Illegal:  Disclaimer in fine print
Legal:    Disclaimer prominent, user must acknowledge
```

### Rule 4: NEVER Promise Returns
```
Illegal:  "Make ₹1L in 6 months"
Legal:    "Results vary; past performance doesn't guarantee future results"
```

### Rule 5: ALWAYS Be Transparent
```
Illegal:  Hide terms and conditions
Legal:    Make terms easy to find and understand
```
