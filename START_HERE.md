## ⚡ QUICK START (5 MINUTES)

### Step 1: Clone Backend
```bash
cd BACKEND
npm install
cp .env.example .env
# Edit .env with your credentials
npm start
```

### Step 2: Clone Frontend
```bash
cd FRONTEND
npm install
# For Android: npm run android
# For iOS: npm run ios
```

### Step 3: Configure Supabase
- Go to [supabase.com](https://supabase.com)
- Create project
- Run db.sql in SQL editor
- Get API keys and add to .env

### Step 4: Set Up OAuth
- Go to Google Cloud Console
- Create OAuth 2.0 credentials
- Add to .env (GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET)

**Result:** Running app with real market data! ✅
