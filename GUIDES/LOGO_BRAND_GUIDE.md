## 🎯 BRAND IDENTITY GUIDELINES

### Logo Usage

**Minimum Size:**
```
App Icon: 192x192 pixels (for Play Store)
Favicon: 32x32 pixels (for website)
Social Media: 400x400 pixels (Instagram, Twitter)
Website Header: 200x200 pixels (recommended size)

Never use smaller than 48x48 for any application
```

**Clear Space (Breathing Room):**
```
Always keep space around logo equal to 20% of logo height
Never place logo directly against edges
Never place on textured backgrounds
Use solid background for clarity
```

**Logo Variations:**
```
1. Full Logo
   - Logo + Text
   - Use for: Website header, marketing

2. Icon Only
   - Just the logo symbol
   - Use for: App icon, favicon, social media

3. Horizontal Layout
   - Logo on left, text on right
   - Use for: Business cards, headers

4. Vertical Layout
   - Logo on top, text below
   - Use for: Website, posters
```

**Color Usage:**
```
✅ Blue + Gold (preferred)
✅ All Blue (if single color needed)
✅ White (on dark backgrounds)
✅ Black (on light backgrounds)
❌ Stretched or distorted
❌ Rotated (only 0° or 90°)
❌ Flipped horizontally
❌ With drop shadows (unless specified)
❌ With distorted colors
```

## 📱 APP ICON DESIGN

### Icon Specifications

**Size Requirements:**
```
iOS:
- 180x180 (iPhone)
- 167x167 (iPad)
- All sizes: Apple Developer Site

Android:
- 512x512 (Play Store listing)
- 192x192 (HDPI phones)
- 144x144 (MDPI phones)
- 96x96 (LDPI phones)

Web:
- 192x192 (favicon)
- 512x512 (desktop shortcut)
```

**Design Rules:**
```
✅ Use core logo element (the W + trend, or mandala+chart)
✅ Keep it simple (recognizable at 48x48px)
✅ Use primary colors (blue + gold)
✅ No text required (should be recognizable without "Stock Wallah")
✅ Rounded corners (20px radius for modern look)
✅ Safe padding around edges
✅ Test at small sizes (to ensure clarity)
```

**Icon Design Steps:**
```
1. Start with your logo
2. Extract the main symbol (no text)
3. Simplify if needed
4. Add rounded corners
5. Place on colored background (#1E90FF blue)
6. Add subtle gold accent if possible
7. Export at all required sizes
8. Test visibility at small sizes
```
