```
4. Contact your bank
5. Try again in 1 hour

```

### "Google Play Account Not Verified"
```
1. Check email (inbox + spam)
2. Look for verification email
3. Click the link
4. Wait 10 minutes
5. Refresh Google Play Console

If not received:
→ Resend verification email
→ Wait 5 minutes
→ Check again
```

### "Can't Create Google Cloud Project"
```
1. Make sure you're signed in with Gmail
2. Check if you have a valid credit/debit card
3. Try using Chrome browser
4. Clear browser cache
5. Try again

If still fails:
→ Try in incognito mode
→ Try different browser
```

### "Claude API Key Not Working"
```
1. Make sure you copied FULL key
2. Make sure no extra spaces
3. Check if API key is enabled
4. Check billing is set up
5. Make test API call

If error about quota:
→ Increase budget in billing settings
```

### "GitHub Repository Won't Create"
```
1. Check username is unique
2. Check repository name is unique
3. Try different name
4. Clear browser cache
5. Try again

If still fails:
→ Try signing out
→ Sign back in
→ Try again
```
